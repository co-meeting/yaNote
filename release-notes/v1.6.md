# yaNote リリースノート v1.6

**リリース日**: 2026-07-18

## 概要

v1.6 は、長らく封印されていたズーム機能の実装と、AI向けエクスポート機能を追加した機能拡張リリースです。

## 主な変更点

### ズーム機能（新機能）

- **Ctrl/Cmd＋マウスホイール**（トラックパッドのピンチ操作を含む）で、カーソル位置を中心に表示を拡大縮小できるようになりました（25%〜200%）
- タッチデバイスでは**2本指のピンチ操作**に対応。指の中間点を中心に拡大縮小し、ピンチ中の平行移動でパンも同時に行えます
- **Ctrl/Cmd+0** で等倍（100%）に戻せます
- ズーム操作中は画面右下に現在のズーム率が一時表示されます
- ズーム率は保存され、次回起動時・共有URL・エクスポートJSONにも引き継がれます
- macOS Safari のトラックパッドピンチ（gesture イベント）にも対応
- v1.2.6 で一度実装を試みて封印した経緯のある機能ですが、当時の失敗原因（ズーム中心の計算誤りと接続線の二重スケール）を特定・解消した上で再実装しました

### AI向けエクスポート機能（新機能）

- コントロールパネルの **AIボタン** から、現在のノートを AI が意図を読み取りやすい Markdown ファイルとして出力できるようになりました
- 座標配置と接続トポロジーから「トピックの島」「カラム対応」「接続の階層」を解析し、アウトライン形式で出力します
- 通常のエクスポート JSON にも、書き手のノード種別・接続線の使い分けを説明する凡例（`semantics`）が埋め込まれます（インポート時には無視されます）

### リリーススクリプトの修正

- 注釈付きタグに変更し、`--follow-tags` で軽量タグが push されない問題を修正しました

## 互換性

- 破壊的変更なし
- 既存のノートデータ（JSON・localStorage・共有URL）との互換性を維持。ズーム率を持たない旧形式データは等倍（100%）として読み込まれます
- Service Worker のキャッシュ名を v1.6 に更新（PWA は自動的に新バージョンへ更新されます）

## ドキュメント更新

- `help.md` — ズーム操作（マウス/トラックパッド・タッチ）の説明を追加、「ズーム機能は実装されていません」の記述を削除
- `docs/yaNote-OperationSpec.md` — キャンバス操作にズームの仕様（範囲・アンカー補正・保存挙動）を追加
- `yaNote_TouchOperationSpec.md` — 2本指ピンチを実装済みとして仕様化、他操作との競合判定を追記
- `README.md` — 使い方にズーム操作を追加
